# LMU Stream Deck Showcase – Quick Start

## What you need

- Le Mans Ultimate
- Elgato Stream Deck
- imported Showcase profile

Important:
In LMU, bind keys in the **Keyboard** column, not in the wheel or controller column.

---

## Setup

### Path A – Express
Fastest option.

1. Back up your current `keyboard.json`
2. Copy `keyboard_preset_showcase.json` into your LMU player folder
3. Rename it to `keyboard.json`
4. Import the Showcase profile into Stream Deck

Warning:
This overwrites your current custom keyboard bindings.

### Path B – Manual
Safer option.

1. Import the Showcase profile into Stream Deck
2. Open LMU: **Settings → Controls → Keyboard**
3. Bind only the functions used in the Showcase layout

## Camera naming note

LMU camera names can be misleading.

| LMU name | Used in Showcase as |
|---|---|
| Swingman Camera | Chase Cam |
| Onboard Cameras | Cockpit Cam |
| Driving Cameras | Dash Cam |
| Tracking Cameras | TV Camera |
| Spectator Cameras | Supporters |

The Showcase uses names based on what the camera actually does.

---

## Good to know

Free Camera works on top of your current view.  
Switch to a camera first, then open the Free Camera folder and activate it.

If you later upgrade to Pro, shared functions use the same LMU key bindings.  
Your setup work is not wasted.
